# ui-design-system

## Objective
Garantir consistência visual moderna e padronizada em toda a aplicação.

## Design Rules

- Utilize espaçamento baseado em escala de 8px (8, 16, 24, 32)
- Bordas arredondadas padrão: 12px
- Sombras suaves: box-shadow leve (ex: 0 2px 8px rgba(0,0,0,0.05))

## Colors
- Primary: #22c55e (verde)
- Background: #f8fafc
- Cards: #ffffff
- Text Primary: #0f172a
- Text Secondary: #64748b

## Typography
- Fonte: Inter ou system-ui
- Títulos: semi-bold
- Corpo: regular

## Behavior
- Nunca usar cores fortes excessivas
- Priorizar legibilidade e hierarquia visual
- Manter estética clean e moderna

## Output
- Interfaces consistentes
- Design profissional estilo SaaS
