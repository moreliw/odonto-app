# responsive-design

## Objective
Garantir responsividade total.

## Breakpoints

- Desktop
- Tablet (1024px)
- Mobile

## Rules

- Grid adaptável
- Sidebar colapsável
- Conteúdo em coluna no mobile

## Output

- Layout funcional em qualquer tela
