# kpi-cards

## Objective
Criar cards de métricas com alto impacto visual.

## Structure

Cada card deve conter:
- Ícone
- Título
- Valor principal
- Variação (%)

## Rules

- Verde = crescimento
- Vermelho = queda
- Layout limpo

## Behavior

- Destaque visual no valor
- Leitura rápida

## Output

- KPIs claros e objetivos
