# dashboard-layout

## Objective
Criar layout padrão de dashboard SaaS moderno.

## Structure

- Sidebar fixa à esquerda (240px)
- Header no topo
- Conteúdo principal com grid

## Layout Composition

Main Content:
- KPI Cards (topo)
- Gráficos
- Tabelas

## Sidebar Rules
- Ícones + labels
- Item ativo destacado
- Navegação clara

## Header Rules
- Barra de busca
- Notificações
- Avatar/perfil

## Behavior
- Layout deve ser responsivo
- Sidebar colapsável no mobile

## Output
- Estrutura organizada
- UX clara e profissional
