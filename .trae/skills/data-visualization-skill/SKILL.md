# data-visualization

## Objective
Criar visualizações de dados modernas e legíveis.

## Libraries
- Recharts ou Chart.js

## Chart Types

- Line Chart (tendência)
- Bar Chart (comparação)
- Donut Chart (proporção)
- Heatmap (atividade)

## Rules

- Evitar poluição visual
- Usar cores suaves
- Incluir tooltips
- Grid leve

## Behavior

- Gráficos responsivos
- Boa leitura de dados

## Output

- Visualização clara
- Experiência profissional
